# Para convertir keystone.png a favicon.ico, puedes usar una herramienta online como https://favicon.io/ o https://convertico.com/
# 1. Sube /public/keystone.png a la herramienta.
# 2. Descarga el favicon.ico generado y reemplaza el archivo /public/favicon.ico en tu proyecto.
# 3. No olvides limpiar la caché del navegador para ver el cambio.
